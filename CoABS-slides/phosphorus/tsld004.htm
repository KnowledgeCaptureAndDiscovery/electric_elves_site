Agent Matching

  • Problem I: topic matching (e.g., interests matcher; e.g., roses)

    • Given: - thousands/millions of agent descriptions
      - a request
    • Find: a set of agents that can fulfill the request
      (and/or something similar to the request)
      (and/or can understand some of the request)
      (and/or could help reformulate the request)

  • Problem II: task-based matching (e.g., activities matcher)

    • Given: - a few dozens/hundreds agent descriptions
      - a request
    • Find: the few agents that can fulfill the request
      (and refinements of it with additional requirements)
